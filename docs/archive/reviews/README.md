# Reviews (historical snapshots)

> [!NOTE]
> These are **frozen, dated review artifacts** — point-in-time adversarial and
> external reviews of specific design-brief sections (most dated 2026-05-28,
> against earlier brief versions). They are preserved for provenance and are
> **not** kept in sync with the current brief or decisions. Relative links to
> `decisions/…` inside these files are written relative to the repository root
> (i.e. `docs/decisions/…`), so they may not resolve when clicked from within
> `docs/reviews/` — resolve them from the repo root. For current state, see
> [`../../README.md`](../../README.md),
> [`../implementation-status.md`](../implementation-status.md), and
> [`../decisions/`](../decisions/).
