# Changelog

Notable changes to Cairn will be documented here, following
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/). The project adopts
semantic versioning at its first release.

## [Unreleased]

Cairn is in **alpha and has not been released** — there are no released
versions yet (see the [README](README.md) status banner). Until the first
signed release:

- Pre-release engineering history lives in the git log and in
  [`docs/implementation-status.md`](docs/implementation-status.md).
- Design-level changes are tracked in the design brief's own changelog
  ([`docs/design-brief.md`](docs/design-brief.md)) and in the architecture
  decision records ([`docs/decisions/`](docs/decisions/)).

The first dated entry here will accompany the first transparency-anchored
release.
